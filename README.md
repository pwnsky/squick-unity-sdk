# Squick Unity SDK

## Intro

The squick Unity SDK

Unity version: 2020.03 above

version: 1.1.0

[Squick Src](https://github.com/pwnsky/squick)



## Feature

Surport TCP or Websocket connection with squick

So you can build you project to web applicaiton.
